# Women Safety — Static Website

A simple, accessible static site to share safety tips, emergency hotlines, and resources.

## What’s included
- index.html — main site
- styles.css — simple responsive styling
- script.js — small JS for mobile nav and contact form handling
- LICENSE — MIT

## Customize
1. Replace the example emergency phone numbers in `index.html`.
2. Replace `support@example.com` in `script.js` with your real support email or integrate a hosted form service (Formspree, Netlify Forms, etc.).
3. Update links in the Resources section.

## Deploy on GitHub Pages
1. Create a repository named `women-safety-website` under your GitHub account `iprceations`.
2. Commit the files and push to GitHub.
3. In repo settings -> Pages, set the source to the `main` branch and `/ (root)`.
4. After a minute, your site will be available at `https://iprceations.github.io/women-safety-website/`.

## Local preview
Open `index.html` in your browser or run a simple static server:
```
# using Python 3
python -m http.server 8000
# then visit http://localhost:8000
```
